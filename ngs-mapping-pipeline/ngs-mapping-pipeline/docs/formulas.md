# Справочник формул NGS Mapping

## Покрытие (Lander-Waterman)
```
C = (N × L) / G
```
C — покрытие (×), N — ридов, L — длина рида, G — размер генома.

## Вероятность пропустить позицию (Пуассон)
```
P(нет покрытия) = e^(−C)
C = 5×  → P ≈ 0.0067  (0.7% пропущено!)
C = 30× → P ≈ 10^(−13) (практически нуль)
```

## MAPQ
```
MAPQ = −10 × log₁₀(P(неверное картирование))
MAPQ=20 → P=1%, MAPQ=30 → P=0.1%, MAPQ=60 → максимум BWA-MEM2
```

## Phred-качество нуклеотида
```
Q = −10 × log₁₀(P(ошибка))
Q=20 → P=1%, Q=30 → P=0.1%, Q=40 → P=0.01%
```

## Вес выравнивания (Нидлман-Вунш / Смит-Вотерман)
```
DP[i][j] = max(
  DP[i−1][j−1] + score(A[i], B[j]),  ← match/mismatch
  DP[i−1][j]   + gap_penalty,         ← gap в seq2
  DP[i][j−1]   + gap_penalty          ← gap в seq1
)
Сложность: O(M × N)
```

## Вычислительная сложность поиска
```
Линейный:        O(N × M)    RAM: O(N)
Бинарный (SA):   O(M × log N) RAM: O(N)
Суффикс. дерево: O(M)        RAM: 64 GB (геном человека)
BWT + FM-index:  O(M)        RAM: 3–4 GB ← используется
```
